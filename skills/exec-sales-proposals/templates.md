# Proposal Deck Quick Reference Templates

## Template A: Standard Service Proposal (15-20 slides)

### Slide Breakdown

1. **Cover Slide**
   - Proposal title: "[Service] Proposal"
   - Client name: "Prepared for: [Client Name]"
   - Your company name + logo
   - Date or contact email

2. **Table of Contents**
   - About [Company]
   - Problem & Opportunity
   - Our Solution
   - Deliverables
   - Process & Timeline
   - Case Studies
   - Investment
   - FAQ
   - Next Steps

3. **About Us**
   - Hero metrics (3-4 large stats in grid)
   - Brief company description
   - Key expertise areas
   - Notable clients (logos)

4. **Problem Statement** (1-2 slides)
   - Section title: "The Challenge" or "Current State"
   - 3-5 specific pain points
   - "What This Costs [Client]" sidebar

5. **Solution Overview**
   - Section title: "Our Approach" or "The Solution"
   - High-level vision
   - 3-5 key components with icons
   - How it addresses the problem

6. **Detailed Deliverables** (2-3 slides)
   - Organized by phase or category
   - Each deliverable:
     - Name
     - Description (1-2 sentences)
     - Purpose/benefit
   - Use icons or numbered list

7. **How We Work**
   - Process visualization (numbered steps)
   - Timeline per phase
   - Collaboration model
   - Support structure

8. **Project Timeline**
   - Visual timeline (horizontal or vertical)
   - Key milestones
   - Duration: "X months from kickoff"

9. **Case Study 1** (If applicable)
   - Client/project name
   - Challenge â†’ Solution â†’ Results
   - Metrics and outcomes
   - Optional: testimonial quote

10. **Case Study 2** (If applicable, similar structure)

11. **Investment**
    - Section title: "Investment" or "Pricing"
    - Total price (large, clear)
    - What's included (breakdown)
    - Payment terms
    - Optional: ROI comparison

12. **FAQ**
    - 4-6 common questions
    - Concise answers
    - Address objections proactively

13. **Next Steps**
    - Clear call to action
    - 2-3 simple steps
    - Contact information
    - Timeline expectation

## Template B: Brief Project Proposal (10-12 slides)

### Slide Breakdown

1. **Cover** - Title, client, company, contact
2. **About** - Credentials and key metrics
3. **Problem** - Challenge statement + impact
4. **Solution** - Approach and key components
5. **Deliverables** - What you'll receive
6. **Process** - How we'll work together
7. **Timeline** - Visual schedule with milestones
8. **Proof** - One strong case study
9. **Investment** - Pricing and what's included
10. **FAQ** - Address top concerns
11. **Next Steps** - Clear action items

---

## Template C: Complex Enterprise Proposal (25-30 slides)

### Slide Breakdown

1. **Cover**
2. **Contents**
3. **Executive Summary** - 1 slide overview
4. **About Our Company** (2 slides)
   - Company overview + credentials
   - Team bios
5. **Understanding Your Challenge** (2 slides)
   - Current state analysis
   - Goals and requirements
6. **Our Solution** (3-4 slides)
   - Strategic approach
   - Technical architecture
   - Integration plan
   - Success metrics
7. **Detailed Deliverables** (3-4 slides)
   - Phase 1 deliverables
   - Phase 2 deliverables
   - Phase 3 deliverables
   - Ongoing support
8. **Methodology** (2 slides)
   - Framework overview
   - Implementation process
9. **Timeline & Milestones** (2 slides)
   - Detailed Gantt chart
   - Key decision points
10. **Case Studies** (3 slides)
    - Case study 1: Similar industry
    - Case study 2: Similar scale
    - Case study 3: Similar challenge
11. **Team & Qualifications** (1-2 slides)
12. **Investment** (2 slides)
    - Pricing breakdown by phase
    - ROI analysis or cost comparison
13. **Risk Mitigation** (1 slide)
14. **FAQ** (1-2 slides)
15. **Next Steps & Timeline**

## Pre-flight Checklist

Before sending your proposal:

**Content**

- [ ] Client name spelled correctly everywhere
- [ ] All dates are current
- [ ] Pricing is accurate
- [ ] No placeholder text (search for: XXXX, TODO, [brackets])
- [ ] Contact information is current
- [ ] All deliverables are listed
- [ ] Timeline is realistic

**Design**

- [ ] Consistent color scheme
- [ ] Consistent fonts
- [ ] All images high quality
- [ ] Text is readable (contrast check)
- [ ] Elements aligned properly
- [ ] Spacing is consistent
- [ ] No overlapping elements

**Business**

- [ ] Value proposition is clear
- [ ] Problem is well articulated
- [ ] Solution addresses problem
- [ ] Differentiators highlighted
- [ ] Social proof included
- [ ] Pricing justified with value
- [ ] Next steps are clear
- [ ] Objections addressed in FAQ

**Polish**

- [ ] Convert to PDF for final delivery
- [ ] File name: "Proposal - [YourCompany] - [ClientName] - [Date].pdf"
- [ ] Test that links work (if any)
- [ ] Check file size (< 10MB ideally)
- [ ] Have someone else review it

## Original Project

Continuous Claude provides session continuity, token-efficient MCP execution, and agentic workflows for Claude Code:

- **Continuity Ledgers** - Preserve state across `/clear` operations
- **Handoffs** - Cross-session context transfer
- **Hooks System** - Intercept Claude Code lifecycle events
- **Skills & Agents** - Reusable workflows for planning, implementation, and debugging

For the latest version, updates, and to contribute, visit the [original repository](https://github.com/parcadei/Continuous-Claude-v2).
